# 📱 PocketGuard

PocketGuard is a simple and focused iOS app that helps you **track your savings, manage financial goals, and stay in control of your money** — without unnecessary complexity.

---

## 🚀 Features

* 💰 **Savings Tracker**
  Monitor how much you’ve saved over time.

* 🎯 **Goal-Based Saving**
  Create savings goals (e.g., phone, travel, emergency fund) and track your progress.

* ➕ **Transaction Logging**
  Easily add deposits and withdrawals.

* 📊 **Progress Visualization**
  See how close you are to reaching your targets.

* 📴 **Offline First**
  Works without internet using local storage.

* ⚡ **Fast & Lightweight**
  Built for speed and simplicity.

---

## 🛠 Tech Stack

* Swift
* UIKit / SwiftUI
* Realm (local database)
* MVVM Architecture

## 🎯 Use Case

PocketGuard is built for:

* Users who want a **simple savings tracker**
* People focused on **goal-based saving**
* Offline-first users who prefer **no cloud dependency**

---

## ⚠️ Limitations

* No bank integration
* No multi-device sync
* Manual transaction input
